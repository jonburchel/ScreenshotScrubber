Screenshot Scrubber Options

Set text values to be replaced on the page and the new replacement values to the list below.
Initial suggested values are provided, but you can customize however you need.
To use the extension, just click the extension icon to replace text or images then.

NOTE:
To replace images, use "img." followed by the image class for Original Text:
  • Browse to a page containing the image to be replaced.
  • Press F12 to open the browser debugger window.
  • Select the Elements tab.
  • Click on the HTML source displayed in the upper left area of the tab.
  • Press Ctrl-F to search within the HTML source.
  • Type <img to search for image tags.
  • Search through all the tags to find the image you need to replace.
  • Use "img." followed by the HTML class attribute value for the image as the Original Text.
  • Use the URL of the replacement image as the New Text.
